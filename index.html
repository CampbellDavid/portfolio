About Me


As a software developer, I have learned that the most important aspect of my job is to create solutions that meet the needs of my clients. I have worked on a variety of projects, and I have always been driven by the desire to create software that is both functional and elegant.

One of my greatest strengths as a software engineer is my ability to work collaboratively with others. I believe that the best software is created through a combination of individual skill and teamwork, and I take pride in my ability to contribute to a team effort. I am always willing to share my knowledge and learn from others, and I believe that this approach has helped me to grow as a developer.

Another important aspect of my work is my commitment to staying current with the latest technologies and techniques. I am always searching for new tools and methodologies that can help me to be more efficient and effective in my work. I believe that this constant learning is essential for staying competitive in the fast-paced world of software development.

Ultimately, my goal as a software engineer is to create software that makes a difference in the lives of others. Whether it is a tool that helps a business to be more productive, or an app that brings joy to its users, I am passionate about using my skills to make a positive impact on the world. I am excited to continue growing as a developer and to take on new challenges as I work to achieve this goal.

Snake Game

Snake One Game built using Vanilla JavaScript.

This was a week-long solo project that taught the basic building blocks of JavaScript and allowed the application of various techniques and methods learned during my first two weeks at General Assembly. The production of the game taught me how to implement array methods, DOM manipulation, function calling, CSS styling and basic HTML rendering.

  • Date: January 2020
  • Repository: git.io/JvKd3
  • Deployed Project: bit.ly/Snake-Game

Breaking Bad

Web application dedicated to the Breaking Bad TV series.

This was a 48-hour pair-project that taught me and my classmate how to render data obtained from an API using React.js. The site we created uses data from an API focusing on the Breaking Bad TV series. Data includes episode summaries, character overviews, memorable quotes and information pertaining to the current live status of any character.

  • Date: January 2020
  • Repository: git.io/JvKds
  • Deployed Project: bit.ly/BreakingBadApp

Inquizitours

Web application for organising pub quiz events.

This was a group project that focused on the development and implementation of an API created by my team. Using MongoDB and Express, we built an API that focuses on various pub establishments around London that hold pub quiz events. My responsibility for this project was to assemble the back end with one other team member. With ten days to complete the task, I was immensely proud with what my team achieved and how we each grew.

  • Date: February 2020
  • Repository: git.io/JvKdZ
  • Deployed Project: bit.ly/InquizitoursApp

Extreme Meet Up

A meet up web application for extreme sports enthusiasts.

A solo project that took a week to develop, using React.js for the front end and Django for the back end. Some structural aspects of the database were inspired from my previous project, as I wanted to implement many of the same user capabilities. The project is designed for extreme sports enthusiasts to meet up for events.

  • Date: March 2020
  • Repository: git.io/JvKdc
  • Deployed Project: bit.ly/XMU-App
