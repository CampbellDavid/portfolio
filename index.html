About Me


Hi there! My name’s David. I’m a full-stack developer from London and a recent graduate of General Assembly’s Software Engineering Immersive.

Prior to GA, I had some novice experience working with software, mainly through self-taught endeavours that entailed editing the code of applications to explore their inner workings. GA has provided an excellent learning opportunity for me, by teaching me the fundamentals of software development and also by allowing me to practice many other soft skills (like public speaking and presentation for example), cultivating a healthy passion for SE in the process. I’m really excited to discover what waits ahead and who I’ll meet along the way!

Outside of work I enjoy exploring new places, cooking exotic foods, and watching TV shows and comedy sketches that focus on political topics. Feel free to reach out if you would like to chat about anything work related or have any great travel tips!

Snake Game

Snake One Game built using Vanilla JavaScript.

This was a week-long solo project that taught the basic building blocks of JavaScript and allowed the application of various techniques and methods learned during my first two weeks at General Assembly. The production of the game taught me how to implement array methods, DOM manipulation, function calling, CSS styling and basic HTML rendering.

  • Date: January 2020
  • Repository: git.io/JvKd3
  • Deployed Project: bit.ly/Snake-Game

Breaking Bad

Web application dedicated to the Breaking Bad TV series.

This was a 48-hour pair-project that taught me and my classmate how to render data obtained from an API using React.js. The site we created uses data from an API focusing on the Breaking Bad TV series. Data includes episode summaries, character overviews, memorable quotes and information pertaining to the current live status of any character.

  • Date: January 2020
  • Repository: git.io/JvKds
  • Deployed Project: bit.ly/BreakingBadApp

Inquizitours

Web application for organising pub quiz events.

This was a group project that focused on the development and implementation of an API created by my team. Using MongoDB and Express, we built an API that focuses on various pub establishments around London that hold pub quiz events. My responsibility for this project was to assemble the backend with one other team member. With ten days to complete the task, I was immensely proud with what my team achieved and how we each grew.

  • Date: February 2020
  • Repository: git.io/JvKdZ
  • Deployed Project: bit.ly/Inquizitours

Extreme Meet Up

A meet up web application for extreme sports enthusiasts.

A solo project that took a week to develop, using React.js for the frontend and Django for the backend.
