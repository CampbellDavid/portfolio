Snake Game

Snake One Game built using Vanilla.js

This was a week-long solo project that taught the basic building blocks of JavaScript and allowed the application of various techniques and methods learned during my first two weeks at General Assembly. The production of the game taught me how to implement array methods, DOM manipulation, function calling, CSS styling and basic HTML rendering.

  • Date: January 2020
  • Repository: git.io/JvKd3
  • Deployed Project: bit.ly/Snake-Game

Breaking Bad

Web application dedicated to the Breaking Bad TV series.

This was a 48-hour pair-project that taught me and my classmate how to render data obtained from an API using React.js. The site we created used data from an API focusing on the Breaking Bad TV series; data included episode synopses, character overviews, memorable quotes and information pertaining to the current live status of said character.

  • Date: January 2020
  • Repository: git.io/JvKds
  • Deployed Project: bit.ly/BreakingBadApp

Inquizitours

Web application for organizing pub quiz events.

This was a group project that focused on the development and implementation of an API developed by my team. Using MongoDB and Express, we developed an API that focused on various pub establishments around London that held pub-quiz events. My focus for this project was to develop the backend with one other team member. With ten days to complete the task, I was immensely proud with what my team achieved and how we each grew.

  • Date: February 2020
  • Repository: git.io/JvKdZ
  • Deployed Project: bit.ly/Inquizitours

Extreme Meet Up

A meet up site for extreme sports enthusiasts.

A solo project that took a week to develop, using React.js for the frontend and Django for the backend. Some structural aspects of the database were inspired from my previous project, as I wanted to implement the same sort of user capabilities. The project is designed for extreme sports enthusiasts to meet up for events.

  • Date: March 2020
  • Repository: git.io/JvKdc
  • Deployed Project: bit.ly/ExtremeMeetUp
